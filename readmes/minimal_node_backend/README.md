## Getting Started <a name = "getting_started"></a>

### Prerequisites

- `node >= 12.0.0`

### Installation

1. Clone this repo locally and install node packages:

```bash
$ git clone https://github.com/librity/repo_name.git
$ cd repo_name
$ yarn install
```

2. Start a dev server:

```bash
$ yarn dev
```

### Testing

Run tests with jest:

```bash
$ yarn test
```
